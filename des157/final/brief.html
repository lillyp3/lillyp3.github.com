Game Design UX Best Practices

Some of the main features that should be considered in mobile game design is UI positions, sliders, and pop-ups. Mobile games have to be designed with the intention of only using a thumb or minimal fingers to perform certain actions. Most buttons or features should be placed in a comfort-to-reach area. Additionally, it should be known when there is additional content on the screen that the user can get to. This can be demonstrated through cutting off the design at the end of the screen, so the user knows they need to slide in that direction. As for pop-ups, it should be obvious when something is not part of the main design. There should be a feature such as a semi-transparent dim or separate animation that signifies a pop-up. Additionally, draggable objects can be an interesting feature; however, it is important to set the anchor point for the design in a place that makes sense when the user touches it with their finger.
